Project Overview

MarketPulse is a retail analytics dashboard designed to help store managers and business analysts visualize and understand sales performance across multiple branches and products. Using interactive charts, the dashboard summarizes sales trends, product performance, branch performance, and unit pricing metrics, enabling data-driven decisions.

The dashboard integrates data from a backend API (FastAPI) and visualizes it using Chart.js and Plotly.js, providing a lightweight and responsive interface.



Technologies Used

Frontend: HTML, CSS, JavaScript

Visualization: Chart.js (Bar, Line, Scatter, Donut), Plotly.js (Area, Box Plot)

Backend: FastAPI serving JSON data

Data: Retail sales records, product line, branch information
